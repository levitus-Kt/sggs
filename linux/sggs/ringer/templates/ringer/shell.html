Если вы перешли на эту страницу, значит, у вас не работает терминал (правая половина окна). Установите shellinabox

Нажимаем на клавиатуре сочетание CTRL + ALT + T (терминал) и вставляем туда команду: sudo apt install shellinabox

После того, как установилось, вставляете: sudo nano /etc/default/shellinabox

Ищете строку SHELLINABOX_PORT, после равно пишете: 4222

Ищете строку SHELLINABOX_ARGS, в кавычках пишете: "--no-beep --disable-ssl -s /:SSH:172.23.56.50"

Должно получиться так: SHELLINABOX_ARGS="--no-beep --disable-ssl -s /:SSH:172.23.56.50"

Нажимаем на клавиатуре сочетание CTRL + S, CTRL + X

Вставляем в терминал:

sudo systemctl restart shellinabox


Возвращаетесь на нужную страницу, в окне вас попросят логин и пароль
